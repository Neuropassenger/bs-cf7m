The plugin allows you to track the operation of Contact Form 7 and sends notifications if new applications are not created.

== Installation ==

1. Upload `bs-cf7m` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set up a custom SMTP server for sending emails in the wp-config.php file.
